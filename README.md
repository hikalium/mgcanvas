mgcanvas
====

mgcanvasはNode-Edge型グラフ構造を容易に編集・表示することを目的としたHTML5アプリケーションです。

あらゆる形式のデータを格納することのできる、ノード型データベースとしての応用も視野に入れて開発を行っています。

## Usage
`www/index.html`をHTML5対応ウェブブラウザで起動するだけです。
